# 手機陀螺儀偵測器

這是一個使用 HTML5 DeviceOrientation API 來偵測手機陀螺儀的網頁應用程式。

## 功能特色

- 📱 即時陀螺儀數據顯示
- 📊 視覺化圖表
- 🔒 HTTPS 支援
- 📱 響應式設計
- 🎨 現代化 UI

## 快速部署到 Netlify

1. 將整個 `TestHtml` 資料夾上傳到 GitHub
2. 前往 [Netlify](https://netlify.com)
3. 點擊 "New site from Git"
4. 選擇您的 GitHub 儲存庫
5. 設定建置命令：留空
6. 設定發布目錄：`TestHtml`
7. 點擊 "Deploy site"

## 本地測試

直接在瀏覽器中開啟 `gyroscope.html` 即可測試。

## 使用說明

1. 在手機上開啟網址
2. 允許瀏覽器存取裝置方向感應器
3. 旋轉手機查看即時數據

## 技術規格

- HTML5 DeviceOrientation API
- CSS3 動畫和漸層
- JavaScript ES6+
- 響應式設計
